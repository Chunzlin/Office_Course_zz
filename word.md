第一章：Word基础入门
1.1 Word 365界面介绍
当你第一次打开Word 365，你会看到这样的界面：

┌─────────────────────────────────────────────────────┐
│ 文件 开始 插入 设计 布局 引用 邮件 审阅 视图 帮助   ← 功能区选项卡
├─────────────────────────────────────────────────────┤
│ ┌─────┐ 字体 段落 样式...                          ← 功能区命令
│ │ 粘贴│                                            ← 快速访问工具栏
│ └─────┘
│ 
│ ┌─────────────────────────────────────────────┐
│ │                                             │ ← 文档编辑区
│ │             这里输入文字                    │
│ │                                             │
│ └─────────────────────────────────────────────┘
│ 
│ 第1页，共1页  字数: 0                           ← 状态栏
│ 中文(中国)   插入   100%                       ← 状态栏信息

主要区域说明：
快速访问工具栏：最左上角，可以添加常用命令

如何添加：点击右边的下拉箭头，勾选需要的命令

开始：最常用的文字编辑功能

引用：目录、脚注等

帮助：获取帮助
文档编辑区：白色区域，在这里输入文字

状态栏：最底部，显示文档信息

1.2 新建、打开和保存文档
新建文档（三种方法）：
启动时新建：

打开Word → 选择"空白文档"
已打开Word时新建：

点击"文件" → "新建" → "空白文档"
快捷键新建：

按 Ctrl + N（同时按住Ctrl键和N键）
保存文档（非常重要！）：
第一次保存：

点击"文件" → "另存为"
选择保存位置（建议：桌面或"文档"文件夹）
输入文件名，例如："我的第一个文档"
点击"保存"
保存格式选择：

.docx：Word标准格式（推荐）
.pdf：只能看不能改的格式
.doc：老版本Word格式
保存快捷键：

Ctrl + S：快速保存（建议每写5分钟按一次）
打开已有文档：
点击"文件" → "打开"
找到文档所在位置
双击文档文件名

五种视图模式：
页面视图（最常用）：

点击"视图"选项卡 → "页面视图"
显示效果和打印出来一样
阅读视图：

点击"视图"选项卡 → "阅读视图"
像看书一样，适合阅读
按ESC键返回编辑
Web版式视图：

显示网页效果
大纲视图：

适合写长文档（如论文）
可以折叠/展开章节
草稿视图：

最简单的编辑界面
不显示图片、页眉页脚
显示比例调整：
方法1：拖动右下角滑块

┌─────────────────────┐
│           - ○ + │ ← 拖动这里
│             100%   │
└─────────────────────┘
方法2：按住Ctrl键滚动鼠标滚轮

第二章：文本编辑基础
2.1 输入与选择文本
输入文字：
在编辑区点击鼠标，出现闪烁光标（|）
直接打字，文字会出现在光标位置
移动光标：
鼠标点击：点哪里，光标到哪里
键盘方向键：上下左右移动
Home键：跳到行首
End键：跳到行尾
Ctrl + Home：跳到文档开头
Ctrl + End：跳到文档结尾
选择文本（重要！）：
鼠标选择：

拖拽选择：按住左键拖动
双击：选中一个词
三击：选中整个段落
在左侧空白处单击：选中一行
在左侧空白处双击：选中一段
在左侧空白处三击：全选整个文档
键盘选择：

Shift + 方向键：逐个选择文字
Ctrl + A：全选整个文档
2.2 复制、剪切和粘贴
复制（三种方法）：
右键菜单：

选中文字 → 右键点击 → 选择"复制"
功能区命令：

选中文字 → "开始"选项卡 → 点击"复制"图标（两张纸的图标）
快捷键：

选中文字 → 按 Ctrl + C
剪切（移动文字）：
选中要移动的文字
按 Ctrl + X 或点击"剪切"图标（剪刀图标）
文字会被暂时"剪掉"
在目标位置按 Ctrl + V 粘贴
粘贴（四种方式）：
粘贴时会出现一个小图标：📋

粘贴选项：
┌─────────────┐
│ 保留源格式   │ ← 保持原来的样子
├─────────────┤
│ 合并格式     │ ← 和周围文字一样
├─────────────┤
│ 只保留文本   │ ← 只要文字，不要格式
└─────────────┘
2.3 查找和替换功能
查找文字：
简单查找：

按 Ctrl + F
右侧出现搜索栏
输入要找的文字
Word会高亮显示所有匹配的文字
高级查找：

"开始"选项卡 → "编辑"组 → 点击"查找"下拉箭头
选择"高级查找"
可以设置更多选项：
区分大小写
全字匹配
使用通配符
替换文字（批量修改）：
例子：把文档中所有的"电脑"改成"计算机"

步骤：

按 Ctrl + H 打开替换对话框
┌─────────────────────────────────┐
│ 查找内容： 电脑                  │
│ 替换为：   计算机                │
├─────────────────────────────────┤
│ [查找下一处] [替换] [全部替换]  │
└─────────────────────────────────┘
在"查找内容"输入：电脑
在"替换为"输入：计算机
点击：
查找下一处：一个一个查看
替换：只改当前找到的
全部替换：一次性全部修改
注意：点击"全部替换"前要确认，因为改错了很难恢复！

第三章：格式设置
3.1 字体格式设置
基础字体设置（在"开始"选项卡）：
┌─────────────────────────────────────────┐
│ 宋体    │   五号    │  B  I  U   abc    │
│ 字体    │   字号    │ 加粗 倾斜 下划线  │
└─────────────────────────────────────────┘
设置方法：

选中要设置的文字
选择字体（常用：宋体、黑体、楷体）
选择字号（数字越大，字越大）
五号 ≈ 10.5磅
小四 ≈ 12磅
四号 ≈ 14磅
快速调整字号：

选中文字后按 Ctrl + ]：增大字号
选中文字后按 Ctrl + [：减小字号
更多字体效果：
点击右下角的小箭头打开字体对话框：

字体对话框：
┌─────────────────────────────────┐
│ 字体         │   高级           │ ← 选项卡
├─────────────────────────────────┤
│ 中文字体：宋体                   │
│ 西文字体：Times New Roman       │
│ 字形：常规 / 倾斜 / 加粗         │
│ 字号：五号                       │
│                                 │
│ 字体颜色：自动                   │
│ 下划线类型：(无)                 │
│ 下划线颜色：自动                 │
│ 效果：删除线 / 上标 / 下标       │
└─────────────────────────────────┘
常用效果：

加粗：标题、重点内容（快捷键：Ctrl+B）
倾斜：强调、引用（快捷键：Ctrl+I）
下划线：重点标记（快捷键：Ctrl+U）
删除线：表示删除的内容
上标：平方米（m²）、化学式（H₂O）
下标：化学式（CO₂）
字体颜色设置：
选中文字
点击"A"图标旁边的下拉箭头
选择颜色
自动：黑色
主题颜色：与文档主题一致
标准色：红、黄、蓝等
其他颜色：自定义颜色
技巧：标题可以用深色，重点可以用红色或蓝色

3.2 段落格式设置
对齐方式：
选中段落后，点击这些图标：

┌─────────────────────┐
│  ≡  ≡  ≡  ≡  ≡     │
│ 左 中 右 两 分散    │
│ 对 对 对 端 对齐    │
│ 齐 齐 齐 对        │
│                     │
│ 快捷键：            │
│ Ctrl+L：左对齐      │
│ Ctrl+E：居中对齐    │
│ Ctrl+R：右对齐      │
│ Ctrl+J：两端对齐    │
└─────────────────────┘
使用场景：

左对齐：正文（默认）
居中对齐：标题、封面文字
右对齐：日期、落款
两端对齐：正式文档、论文
行距设置：
快速设置：

选中段落
点击"行和段落间距"图标（四条横线带箭头）
选择：1.0、1.15、1.5、2.0等
精确设置：

选中段落
右键 → "段落"
在"行距"中选择：
单倍行距：标准
1.5倍行距：阅读舒服
2倍行距：老师批改
固定值：精确控制，如20磅
最小值：至少指定的值
建议：

普通文档：1.5倍行距
论文、报告：固定值20-24磅
段落缩进：
首行缩进（中文习惯）：

选中段落
右键 → "段落"
特殊格式 → "首行缩进"
值：2字符（标准）
快速缩进方法：

Tab键：光标在段落开头时，按Tab键缩进
增加缩进按钮：点击增加缩进量图标
段前段后间距：
让段落之间更有层次感

选中段落
右键 → "段落"
设置：
段前：0.5行（标题下方空一点）
段后：0.5行（段落下方空一点）
3.3 项目符号和编号
项目符号（无顺序列表）：
例如购物清单：
• 苹果
• 香蕉
• 橙子
添加方法：

输入文字后，点击"项目符号"图标（三个点带箭头）
或者输入"*"加空格，自动变成项目符号
更改样式：

点击项目符号旁边的下拉箭头
选择喜欢的样式：
圆点 •
方框 □
箭头 →
自定义图片
编号（有顺序列表）：
例如操作步骤：
1. 第一步
2. 第二步
3. 第三步
添加方法：

输入文字后，点击"编号"图标（1. 2. 3.）
或者输入"1."加空格，自动变成编号
多级编号（大纲式）：

例如：
第一章
  1.1 第一节
    1.1.1 第一小节
设置方法：

点击"多级列表"图标
选择带级别的样式
按Tab键：降低级别
按Shift+Tab：提升级别
第四章：页面布局与设计
4.1 页面设置
纸张大小设置：
常用纸张：

A4：最常用，打印文档、报告
A3：海报、宣传页
16开：书籍
Letter：国外常用
设置步骤：

"布局"选项卡 → "纸张大小"
选择尺寸
如需自定义：点击"其他纸张大小"
页边距设置：
页边距是页面四周的空白区域

预设边距：

"布局"选项卡 → "页边距"
选择：
普通：上下2.54cm，左右3.18cm（默认）
窄：所有边1.27cm（节约纸张）
适中：左右2.54cm
宽：所有边2.54cm（适合装订）
自定义边距：

点击"自定义边距"
设置：
上、下、左、右的数值
装订线位置和宽度
纸张方向：
纵向：正常文档（默认）
横向：表格、宽图
设置："布局" → "纸张方向" → 选择纵向/横向

分栏：
让文档像报纸一样分栏排列

设置：

选中要分栏的文字
"布局" → "栏"
选择：
一栏（默认）
两栏
三栏
偏左（左窄右宽）
偏右（左宽右窄）
4.2 页眉、页脚和页码
页眉设置：
页眉在页面顶部，常用于显示文档标题、公司名称等

添加页眉：

"插入"选项卡 → "页眉"
选择样式（空白、边线型、传统型等）
在页眉区域输入文字
常见页眉内容：

文档标题
章节名称
公司Logo
作者姓名
页脚设置：
页脚在页面底部，常用于显示页码、日期等

添加页脚：

"插入"选项卡 → "页脚"
选择样式
在页脚区域输入内容
页码设置：
添加页码：

"插入" → "页码"
选择位置：
页面顶端
页面底端（最常用）
页边距
当前位置
选择样式：
普通数字：1, 2, 3
带格式：第1页，-1-
罗马数字：Ⅰ, Ⅱ, Ⅲ
页码格式设置：

双击页脚进入编辑
"页眉和页脚工具" → "设计"选项卡
点击"页码" → "设置页码格式"
设置：
编号格式：1, 2, 3 或 一, 二, 三
起始页码：从第几页开始
首页不同：
封面通常不需要页码

设置方法：

双击页眉/页脚
勾选"首页不同"
首页的页眉页脚可以留空
奇偶页不同：
书籍、论文常用，奇数页和偶数页显示不同内容

设置方法：

双击页眉/页脚
勾选"奇偶页不同"
分别设置奇数页和偶数页的页眉页脚
4.3 分页与分节
分页（强制换页）：
什么时候用：

新章节从新一页开始
表格、图片需要单独一页
封面、目录、正文分开
三种方法：

快捷键：光标位置按 Ctrl + Enter
功能区："插入" → "分页"
布局选项卡："布局" → "分隔符" → "分页符"
查看分页符： 点击"开始" → "段落"组的"显示/隐藏编辑标记"（¶图标） 分页符显示为：---------分页符---------

分节符（高级功能）：
分节可以让文档不同部分有不同的格式

四种分节符：

下一页：新节从下一页开始（最常用）
连续：新节从同一页开始
偶数页：新节从下一个偶数页开始
奇数页：新节从下一个奇数页开始
使用场景：

例：毕业论文结构
┌───────────────────────┐
│ 封面                  │ ← 第一节
├───────────────────────┤
│ 摘要                  │ ← 第二节（页眉不同）
├───────────────────────┤
│ 目录                  │ ← 第三节（页码用罗马数字）
├───────────────────────┤
│ 正文                  │ ← 第四节（页码用阿拉伯数字）
└───────────────────────┘
设置不同节的页码：

在需要改变格式的位置插入分节符
双击第二节的页脚
点击"链接到前一条页眉"取消链接
单独设置第二节的页码格式
第五章：插入对象

插入图片：
三种方法：

直接插入：

"插入"选项卡 → "图片"
选择"此设备"（从电脑选择）
找到图片文件 → 插入
复制粘贴：

在网页或文件夹中复制图片
回到Word按 Ctrl + V 粘贴
拖拽插入：

打开图片所在文件夹
直接拖拽图片到Word文档中
图片格式调整：
选中图片后，会出现"图片格式"选项卡

大小调整：

拖动四周的圆点：等比例缩放
拖动边上的圆点：拉伸变形
精确设置：在"大小"组输入高度宽度
图片样式：

选中图片
"图片格式"选项卡 → "图片样式"
选择：
简单边框
阴影效果
立体效果
柔化边缘
图片位置：

选中图片
点击"位置"下拉箭头
选择文字环绕方式：
嵌入型：像文字一样（默认）
四周型：文字环绕图片
紧密型：文字紧贴图片
衬于文字下方：图片做背景
浮于文字上方：图片盖住文字
插入形状：
步骤：

"插入" → "形状"
选择形状类别：
基本形状：矩形、圆形
箭头：各种箭头
流程图：流程图形状
星与旗帜：五角星、旗帜
编辑形状：

选中形状
"形状格式"选项卡
可以设置：
填充颜色
轮廓颜色
阴影效果
三维旋转
组合多个对象：
当有多个图片、形状需要一起移动时

组合方法：

按住Ctrl键，依次点击每个对象
右键 → "组合" → "组合"
取消组合：右键 → "组合" → "取消组合"
5.2 插入表格
创建表格：
三种方法：

快速插入：

"插入" → "表格"
移动鼠标选择行列数（最大10×8）
插入表格对话框：

"插入" → "表格" → "插入表格"
输入行数、列数
例如：5行3列
绘制表格：

"插入" → "表格" → "绘制表格"
鼠标变成铅笔，可以自由绘制
适合不规则表格
表格基本操作：
选择表格元素：

点击单元格：选中一个单元格
点击表格左上角十字箭头：选中整个表格
在左侧/上侧边缘移动鼠标：选中整行/整列
调整大小：

拖动表格线：调整行高列宽
拖动右下角小方块：整体缩放
添加/删除行列：

右键点击表格
选择：
插入：在左侧/右侧/上方/下方插入
删除：删除行/列/单元格
表格样式设置：
快速样式：

点击表格任意位置
"表格设计"选项卡 → "表格样式"
选择预设样式（有颜色、边框等）
边框和底纹：

选中单元格或整个表格
"表格设计" → "边框"
选择：
所有框线
外侧框线
内部框线
无框线
底纹（背景色）：

选中单元格
"表格设计" → "底纹"
选择颜色
表头用深色
隔行用浅色（提高可读性）
表格数据操作：
对齐方式： 选中单元格 → "表格布局"选项卡 → 对齐方式

靠上两端对齐
水平居中（最常用）
靠下右对齐
合并单元格：

选中要合并的多个单元格
右键 → "合并单元格"
用途：制作大标题单元格
拆分单元格：

选中单元格

第九章：常见问题与解决
9.1 常见错误处理
问题1：文字无法对齐
现象：行尾文字参差不齐

解决：

选中段落
"开始" → "段落" → 对齐方式选择"两端对齐"
或者按 Ctrl + J
问题2：图片无法移动
现象：图片像文字一样固定在某个位置

解决：

选中图片
"图片格式" → "排列" → "环绕文字"
选择"四周型"或"紧密型"
问题3：页眉横线删不掉
现象：页眉有一条横线，无法删除

解决：

双击进入页眉编辑
选中段落标记（¶）
"开始" → "边框" → "无框线"
问题4：空格变成圆点
现象：文档中出现很多小圆点

解决：

"开始" → "段落"组
点击"显示/隐藏编辑标记"（¶图标）
这是格式标记，打印时不会出现
问题5：文字突然变大/变小
现象：选中文字时不小心改变了大小

解决：

按 Ctrl + Z 撤销
或者选中文字，重新设置字号
问题6：文档损坏打不开
解决：

打开Word程序
"文件" → "打开" → "浏览"
选择损坏文档
点击"打开"下拉箭头 → "打开并修复"
